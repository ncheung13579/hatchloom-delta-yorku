# Hatchloom Frontend

React + TypeScript frontend for Screens 300-303 (School Admin Dashboard, Experiences, Experience Detail, Enrolment).

See the [root README](../README.md) for full setup instructions, demo walkthrough, and architecture docs.

## Quick Reference

```bash
npm install     # Install dependencies
npm run dev     # Start dev server on port 3000
npm run build   # Type-check and production build
npm test        # Run 135 tests (Vitest)
```

Requires backend services running on ports 8001-8003. Start them with `docker compose up -d` from the repo root.
